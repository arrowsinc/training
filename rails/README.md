## 考え方
* DRY
	* Don't Repeat Yourself.
	* 3回同じコードを書いたら共通化する
* CoC
	* Convention over Configuration 
	* 設定より規約
* YAGNI
	* You ain't gonna need it
	* 機能は実際に必要となるまでは追加しない
